# Video Usage Instructions

To use your own local videos instead of internet URLs, follow these simple steps:

1. Place your `.mp4`, `.webm`, or other video files directly into this `public` folder.
   *Example: You copy a file named `my-bank-video.mp4` directly into `public/`.*

2. Go to the Admin Panel on the digital signage display.
3. In the **Video URL** field, simply type a forward slash `/` followed by the exact filename.
   *Example: `/my-bank-video.mp4`*

4. Save the settings. Your video will seamlessly load and loop continuously.

### Important Note on Looping:
For a perfectly smooth "invisible" loop without any stutter or black flashes:
- Make sure your video file is edited cleanly so the last frame exactly matches the first frame.
- Avoid videos with built-in fade-to-black effects at the end unless it's an intended visual effect.
